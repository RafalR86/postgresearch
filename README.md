# Postgresearch

## About <a name = "about"></a>

The purpose of the project is to check postgresql advantages and disadvantages as a search service and to check the best software architecture choices.

### Prerequisites

```
- .NET Core SDK 3.1.100
- PostgreSQL 11 or higher
```

### Installation

```
- Create your appsettings file based on appsettings.Development.json.template file, e.g. appsettings.Development.json for development environment
- Change connection string in appsettings file
```
